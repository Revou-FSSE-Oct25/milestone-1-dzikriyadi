## ğŸš€ Deployment (Menggunakan GitHub Pages)

Proyek ini dideploy menggunakan **GitHub Pages** karena merupakan situs statis (_static website_). Berikut adalah langkah-langkah deployment singkat:

1.  **Buat Repository:** Pastikan seluruh kode sudah di-_push_ ke _repository_ GitHub Anda (`revou-fsse-oct25/milestone-1-dzikriyadi`).
2.  **Akses Pengaturan:** Navigasi ke tab **Settings** pada repository Anda.
3.  **Pilih Pages:** Di sidebar kiri, klik **Pages**.
4.  **Konfigurasi Sumber:**
    - Ubah _Source_ menjadi **Deploy from a branch**.
    - Pilih _Branch_ utama proyek (biasanya `main` atau `master`).
    - Pilih folder `/ (root)` untuk direktori utama file website.
5.  **Simpan:** Klik **Save**.
6.  **Akses Link:** GitHub Pages akan memproses _build_ dan menyediakan _live link_ dalam beberapa menit. Berikut Link Deploynya `https://revou-fsse-oct25.github.io/milestone-1-dzikriyadi/`.
